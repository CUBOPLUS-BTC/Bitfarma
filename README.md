💊 ¿Qué es BitFarma?
BitFarma es una aplicación web para la gestión integral de una farmacia moderna, 
donde los usuarios pueden cargar sus recetas o prescripciones médicas de forma digital para que el farmacéutico las prepare,
y gestione la entrega, ya sea retiro en tienda o envío a domicilio.
Lo que hace única a BitFarma es su integración con Bitcoin como medio de pago, 
combinando la soberanía financiera con la soberanía sobre tu salud.
"Porque tu salud y tu dinero te pertenecen."
